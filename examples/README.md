# Examples

Ready-to-use Prompt-Armor blocks. Copy any file's content and paste it directly into ChatGPT, Claude, or any other AI chat.

| File | Description |
|------|-------------|
| `landing-page.prompt-armor` | Generates a modern SaaS landing page for "CloudSync" with Hero, Features, Pricing, and Testimonials |
| `api-backend.prompt-armor` | Creates a REST API with JWT authentication and SQLite for a task management app |
| `email-template.prompt-armor` | Writes a professional B2B cold outreach email template in German |

## Test the tamper protection

1. Copy the content of any `.prompt-armor` file
2. Paste it into an AI chat -- it will execute the hidden prompt
3. Now change **one character** inside the `ARMOR BLOCK` section
4. Paste it again -- the AI will refuse execution

That is Prompt-Armor in action.
