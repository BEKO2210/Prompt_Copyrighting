Prompt-Armor Runner

Server-side Execution mit OpenAI

[ON] Prompt bleibt auf dem Server
So funktioniert es:
1. Wähle einen gespeicherten Prompt aus
2. Fülle die Variablen aus (Thema, Keyword, etc.)
3. Klicke "Ausführen" - der Prompt wird auf dem Server an OpenAI gesendet
4. Du siehst nur die Antwort, niemals den Prompt-Text!
